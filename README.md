# NeuroTools
A Python Toolbox for Statistics and Signal Processing (EEG and bio).

---

## Description

Features:

- EEG (based on [mne](http://martinos.org/mne/stable/index.html))
- EDA

## Install

Run the following:

```bash
pip install https://github.com/neuropsychology/NeuroTools.py/zipball/master
```
